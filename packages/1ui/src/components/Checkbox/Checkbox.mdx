## Accessibility

The Checkbox component is built with accessibility in mind. It uses the native `<input type="checkbox">` element, ensuring proper keyboard navigation and screen reader support.

## Customization

The Checkbox component can be customized using Tailwind CSS classes. You can modify its appearance by passing a `className` prop or by adjusting the default styles in the component definition.
