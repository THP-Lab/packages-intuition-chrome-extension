# PageHeader

A header component for pages that displays a title, optional subtitle, and last updated timestamp.

## Usage

```tsx
import { PageHeader } from '@1ui/components'

export function Page() {
  return (
    <PageHeader
      title="Network"
      subtitle="View and manage your network connections"
      lastUpdated="3s"
    />
  )
}
```
